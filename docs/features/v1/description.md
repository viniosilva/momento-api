# Momento — Release v1 (MVP)

## Scope

Users must be able to:

### Accounts
- Sign up
- Authenticate

### Events
- Create private events
- Edit events
- Archive events
- View upcoming events
- View past events (memories)

### Invitations
- Invite people
- Accept invitations
- Decline invitations
- List participants

### Photos (Core Value)
- Upload photos to events
- View event photo gallery
- Only invited users can view photos

If this works well, the product already delivers core value.

---

## Out of Scope (v1)

Explicitly NOT included:

- Videos
- Public events
- Public event discovery/search
- Photo comments
- Likes/reactions
- Chat
- Advanced notifications
- Native mobile apps
- Google Photos / iCloud integrations (future)

---

## Initial Metrics

- Average number of photos per event  
- % of invited users who upload photos  
- Events created per user  
- Weekly active users  

Goal: validate the shared album behavior.

---

## Assumptions

- Users want a private album per event
- Simple upload experience is critical

---

## Risks

- Onboarding and invitation friction
- Competing with messaging apps and cloud drives
- Future need for export and integrations

---

## Definition of Success

Groups consistently use Momento to store photos from real events.
