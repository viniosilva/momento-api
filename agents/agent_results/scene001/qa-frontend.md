## Cenários de Sucesso
- Login com Sucesso: Deve direcionar o usuário para a página inicial ao inserir credenciais válidas (telefone e senha).

## Cenários de Erro
- Telefone Vazio: Deve retornar erro quando o número de telefone for vazio.
- Formato de Telefone Não Reconhecido: Deve retornar erro quando o número de telefone não estiver no formato correto.
- Telefone Já Cadastrado: Deve retornar erro quando o telefone já estiver cadastrado no sistema.
- Senha Inválida: Deve retornar erro quando a senha estiver inválida.
- Senha Vazia: Deve retornar erro quando a senha for vazia.
- Erro Interno: Deve retornar erro quando houver falha ao tentar inserir o usuário no banco de dados.
