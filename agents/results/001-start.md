# Product Management: Aplicativo de Notas Colaborativas (V1)

## Necessidade de Negócio
Como **usuário de um aplicativo de notas**, eu quero **escrever e compartilhar conteúdos de forma colaborativa** para que **eu possa organizar projetos e estudos com eficiência e segurança**.

---

## Cenários de Entrega (Backlog V1)
*Seguindo o princípio INVEST: Unidades independentes e com valor de negócio.*

### 1. Autenticação e Acesso
- `Como um novo usuário, eu quero criar uma conta para iniciar o uso da plataforma.`
- `Como um usuário, eu quero realizar o login com e-mail e senha para acessar meu ambiente personalizado.`
- `Como um usuário, eu quero recuperar minha senha via e-mail para retomar o acesso em caso de esquecimento.`

### 2. Gestão de Conteúdo
- `Como um usuário, eu quero registrar textos em novas notas para capturar ideias imediatamente.`
- `Como um usuário, eu quero visualizar o conteúdo de notas existentes para consulta de informações.`
- `Como um usuário, eu quero modificar o texto de notas salvas para manter a informação atualizada.`
- `Como um usuário, eu quero listar todos os meus registros para ter uma visão panorâmica da minha produção.`
- `Como um usuário, eu quero arquivar registros inativos para organizar meu espaço de trabalho sem excluir dados.`

### 3. Colaboração e Permissões
- `Como um usuário, eu quero gerar links de compartilhamento para permitir o acesso de terceiros à minha nota.`
- `Como um usuário, eu quero definir permissões de "apenas leitura" para proteger o conteúdo contra alterações indesejadas.`
- `Como um usuário, eu quero definir permissões de "edição" para permitir a escrita colaborativa em tempo real.`
- `Como um usuário, eu quero configurar um prazo de expiração para os links compartilhados para garantir a privacidade temporária dos dados.`
